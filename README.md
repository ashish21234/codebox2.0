# 📦 CodeBox - AI-Powered Coding Adventure

CodeBox is a modern, AI-integrated learning platform designed to help beginners start their coding journey through interactive courses, real-world projects, and AI-assisted study materials.

![Banner](/public/hero1.gif)

## 🚀 Features

- **Interactive Courses**: Carefully structured coding courses with chapters and hands-on exercises.
- **AI-Powered Learning**: Get extra study materials and exercises generated by AI (Google Gemini) tailored to each course.
- **In-Browser Code Editor**: Write and run code directly in your browser using integrated **Sandpack** and **CodeMirror** editors.
- **User Dashboard**: Track your progress with XP points, streaks, and a visual representation of your learning journey.
- **Project Sandbox**: Create, save, and manage your own coding projects within the platform.
- **Gamified Experience**: A unique "pixel-art" themed UI that makes learning to code feel like an adventure.
- **Authentication**: Secure login and user management powered by **Clerk**.
- **Responsive Design**: Built with **Tailwind CSS 4** for a seamless experience across all devices.

## 🛠️ Tech Stack

- **Framework**: [Next.js 19](https://nextjs.org/) (App Router)
- **Styling**: [Tailwind CSS 4](https://tailwindcss.com/)
- **UI Components**: [Radix UI](https://www.radix-ui.com/) & [Lucide Icons](https://lucide.dev/)
- **Authentication**: [Clerk](https://clerk.com/)
- **Database**: [Neon PostgreSQL](https://neon.tech/)
- **ORM**: [Drizzle ORM](https://orm.drizzle.team/)
- **AI Engine**: [Google Gemini Pro](https://deepmind.google/technologies/gemini/)
- **Code Execution**: [Sandpack](https://sandpack.codesandbox.io/)
- **State Management**: React Hooks & Context API

## ⚙️ Getting Started

### Prerequisites

- Node.js (Latest LTS)
- NPM or PNPM
- A Neon Database account
- A Clerk account
- A Google AI (Gemini) API Key

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/your-username/codebox.git
   cd codebox
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Set up environment variables:**
   Create a `.env` file in the root directory and add the following:
   ```env
   NEXT_PUBLIC_CLERK_PUBLISHABLE_KEY=your_clerk_pub_key
   CLERK_SECRET_KEY=your_clerk_secret_key
   NEXT_PUBLIC_CLERK_SIGN_IN_URL=/sign-in
   NEXT_PUBLIC_CLERK_SIGN_UP_URL=/sign-up

   DATABASE_URL=your_neon_postgres_url

   GEMINI_API_KEY=your_google_gemini_api_key
   
   GOOGLE_SEARCH_API_KEY=your_google_search_key
   GOOGLE_SEARCH_ENGINE_ID=your_search_engine_id
   ```

4. **Sync the database:**
   ```bash
   npx drizzle-kit push
   ```

5. **Run the development server:**
   ```bash
   npm run dev
   ```

Open [http://localhost:3000](http://localhost:3000) to see your app in action!
